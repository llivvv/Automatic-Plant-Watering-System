# Automatic-Plant-Watering-System

## Wiring: 

#### Sensor: 
- A0: Green --> pin A0
- VCC: White --> pin 5V
- GND: Blue --> pin GND



#### Pump: 
- Black --> (-) rail
- Red --> Relay NO

#### Relay: 
- DC+: Orange --> (+) rail
- DC-: Blue --> (-) rail
- IN: Yellow --> pin 7
- COM: White --> (+) rail
